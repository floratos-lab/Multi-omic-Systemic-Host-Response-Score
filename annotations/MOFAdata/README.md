# Overview

Files in this folder were downloaded from the MOFAdata GitHub project:

-   <https://github.com/bioFAM/MOFAdata/tree/master/data>

These are the original files, specifying gene sets using Ensembl ids; we convert them to versions keyed on NCBI gene ids, using the method:

-   src/convertMOFAdata.R::convertMOFAGeneSetData()
