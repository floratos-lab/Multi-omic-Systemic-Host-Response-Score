# MoSS-src/ — External MoSS Scoring

This folder provides a self-contained toolkit for estimating MoSS values from
bulk whole-blood RNA-seq data using the 14-gene linear model trained in the
RESERVE-U-2-TOR (Tororo discovery cohort).

In the discovery cohort, the model closely reproduced MoSS values generated by
the de novo multi-omic MOFA+ approach (training R² = 0.996; RMSE = 0.099).
When applied to RESERVE-U-1-EBB (Entebbe validation cohort), predicted
values remained strongly concordant with de novo MOFA+ scores (Pearson r =
0.973; external R² = 0.947), confirming that this compact transcriptomic
signature recapitulates the multi-omic structure underlying MoSS.

---

## Scripts

### `validation_rfe_external.R`
Core validation framework providing all scoring and gene ID harmonization
functions used by `run_validation()`. **Sourced automatically; not run
directly.**

Functions defined:
- `load_validation_data(config)` — dispatches to format-specific loaders
  (CSV, TSV, Excel, GEO accession, RDS)
- `standardize_gene_ids(counts, id_type)` — strips Ensembl version suffixes,
  resolves duplicate row IDs by highest mean expression
- `map_model_genes_enhanced(model_coef_df, target_id_type, count_gene_ids)` —
  maps model gene symbols and Entrez IDs to the validation cohort's ID space
  with multi-stage fallback strategies
- `create_validation_coefficients(mapping_results, model_coefficients)` —
  assembles the final coefficient table in the target ID space
- `preprocess_rnaseq_mofa_style(raw_mat, top_n, scale_views, keep_genes)` —
  VST normalization, variance filtering, optional z-scoring
- `run_validation(validation_config, model_coefficients, output_dir)` —
  main entry point; returns per-sample MoSS predictions, QC plots, and gene
  mapping reports

### `methods.R`
Shared preprocessing utilities. Sourced automatically by
`validation_rfe_external.R`; not run directly.

**Input:**
- Raw RNA-seq expression matrix file (gene-level read counts)
- Must contain all 14 genes in the model: MARCHF2, LINC00861, RALB, LRP10, IL17RA, GMPR, GLT1D1, RAB32, MCTP1, NFIL3, TCF7, ALOX5, MXD1, FCMR.

**Outputs (written to user-specified `output_dir`):**
- {cohort}_gene_mapping.csv
- {cohort}_qc.png
- {cohort}_moss_distribution.png
- {cohort}_moss_predictions.csv
- {cohort}_final_coefficients.csv
- {cohort}_summary.csv
- {cohort}_validation_log.txt (if save_log = TRUE)

## Applying the Model to a New Cohort

### Quick start

Copy the usage template below into a new R script (or use it interactively).
Set `cohort_config` to point to your data, then call `run_validation()`.

```r
#################
# DO NOT MODIFY #
#################
library(here)
source(here("MoSS-src", "validation_rfe_external.R"))

model_coefficients <- data.frame(
  gene_symbol = c("(Intercept)",
                  "MARCHF2", "LINC00861", "RALB", "LRP10", "IL17RA",
                  "GMPR", "GLT1D1", "RAB32", "MCTP1", "NFIL3",
                  "TCF7", "ALOX5", "MXD1", "FCMR"),
  entrez_id   = c("(Intercept)",
                  "51257", "100130231", "5899", "26020", "23765",
                  "2766", "144423", "10981", "79772", "4783",
                  "6932", "240", "4084", "9214"),
  coefficient = c(-3.43160568019838E-05,
                   0.147715633572941, -0.183731392712583, -0.00309730715148706,
                   0.22906810645259,   0.124647212894111,  0.081720720008676,
                   0.1372385028354,    0.164788504687379,  0.13763710449136,
                   0.134859140813481, -0.152256683393997,  0.117062936053482,
                   0.123687359805163, -0.0555303398048305),
  stringsAsFactors = FALSE
)

###############
# MODIFY HERE #
###############
# @cohort_name        name assigned to the cohort; used for output file naming
# @data_source        input format: "csv", "tsv", "excel", "rds", or "geo"
# @file_path          path to the RNA-seq expression matrix file (in the form of gene-level read counts)
# @gene_id_column     column containing gene IDs (column number or name string)
# @count_start_column first column of count data (column number or name string)
# @gene_id_type       gene ID type in the input: "ensembl", "entrez", or "symbol"

cohort_config <- list(
  cohort_name        = "Cohort",
  data_source        = "csv",
  file_path          = "file/path/to/data.csv",
  gene_id_column     = 1,
  count_start_column = 2,
  gene_id_type       = "ensembl"
)

cohort_results <- run_validation(
  validation_config  = cohort_config,
  model_coefficients = model_coefficients,
  output_dir         = "path/to/results/directory"
)
```

### Supported input formats

| Format | `data_source` value |
|---|---|
| CSV | `"csv"` |
| TSV / TSV.GZ | `"tsv"` |
| Excel | `"excel"` |
| RDS | `"rds"` |
| GEO accession | `"geo"` |

### Supported gene ID types

| ID type | `gene_id_type` value |
|---|---|
| Ensembl | `"ensembl"` |
| Entrez | `"entrez"` |
| Gene symbol | `"symbol"` |

### Expected input format

A gene × sample expression matrix (gene-level read counts) with one gene ID column followed by sample
count columns:

| ensembl_gene_id | sample_1 | sample_2 |
|---|---|---|
| ENSG00000121410 | 90 | 40 |
| ENSG00000268895 | 40 | 10 |

## Derivation of the MoSS Prediction Model (for reproducibility)

The 14-gene model was derived from the full discovery and validation pipeline
in `src/`. The scripts below are provided for transparency; they do not need to
be re-run to generate MoSS predictions in a new cohort.

- `src/validation_lasso.R` / `src/validation_lasso_review.R` — LASSO feature
  pre-selection from RNA-seq and proteomics views
- `src/validation_rfe_feature_selection.R` — Recursive Feature Elimination (RFE) to select the final minimal
  gene and protein signatures
